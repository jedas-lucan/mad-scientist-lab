Butcherman Caps

Designer : Ray Larabie
Year : 2001
Features :
날카롭고 뾰족하며, 핏자국이 떨어지는       듯한 디테일.
광기, 공포, 위험을 보여주는 강렬한 스타일.
실험실의 실패나 폭발적 에너지를 암시.

Unlock


Designer :Denis Masharov
Year : 2011
Features :
깔끔한 직선과 라운드된 코너의 조화.
잠겨진 문을 풀거나 비밀을 암시하는 듯한 디자인.
단순하면서도 과학적이고 실험적인 느낌 전달.

Henny Penny


Designer : Brownfox
Year : 2011
Features :
곡선이 강조된 굵은 글자체.
활기차고 다이내믹한 느낌으로 에너지를 더함.
다소 희극적이면서도 혼란스러운 감성    표현.

Eater


Designer : Ray Larabie
Year : 2009
Features :
거친 질감과 깔끔하지 않은 라인.
글자 끝부분이 삐져나온 느낌.
긴박하고 무서운 분위기를 조성하며,          매드사이언티스트의 어두운 실험                      분위기에 적합.

Metal Mania


Designer : Ray Larabie
Year : 2011
Features :
각진 끝부분과 뾰족한 디테일.
두꺼운 스트로크로 강렬하고 공격적인 분위기.
광기와 힘을 동시에 전달하는 디자인.

Rubik Beastly


Designer : Hubert and Fischer
Year : 2015
Features :
루빅 폰트의 깔끔하고 기하학적인 모양에 날카로운 디테일과 왜곡된 요소를 추가.
글자 끝부분이  뾰족한 느낌으로 거칠고 과감한 분위기.

Irish Grover


Designer : Sideshow
Year : 2011
Features :
둥글둥글한 형태에  왜곡으로 레트로한      느낌을 줌.
친근하지만 약간의 기괴한 분위기를 강조.
만화적인 스타일.

Griffy


Designer : Neapolitan
Year : 2011
Features :
글자마다 다른 두께와 크기로 자연스럽고 비정형적.
손으로 쓴 듯한 느낌이 인간적인                             매드사이언티스트의 감성을 더함.
즉흥적이고 창의적인 분위기 강조.
